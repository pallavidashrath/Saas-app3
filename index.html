Design System

This step establishes the visual foundation for the entire application. All components must follow these strict guidelines to maintain consistency.

Design Token
--color-accent: #8B0000

Spacing Scale

Only use these values:

  • • 8px — Small gaps
  • • 16px — Standard spacing
  • • 24px — Section padding
  • • 40px — Large margins
  • • 64px — Page sections

Interaction Rules

  • • Transitions: 150–200ms
  • • Easing: ease-in-out
  • • No bounce effects
  • • No parallax scrolling
  • • Subtle hover states only

Button Usage

Primary: Main actions only
Secondary: Cancel, back, less important actions
